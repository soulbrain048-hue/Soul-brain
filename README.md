here| Problem                                   | Solution                                                                     |
| ----------------------------------------- | ---------------------------------------------------------------------------- |
| `apt` mein tools nahi mil rahe            | **Auto-detection** system - pehle check karta hai tool installed hai ya nahi |
| Missing package errors                    | **Install guide** dikhata hai har tool ke liye alag command                  |
| Go tools (subfinder, amass, nuclei, etc.) | `go install` se install honge                                                |
| Ruby tools (wpscan)                       | `gem install` se                                                             |
| Binary tools (aquatone, trivy)            | Direct download se                                                           |
